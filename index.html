Why Obsidian

Built like a premium storefront, not a flat landing page.

The system is designed to feel layered and advanced, with reusable glass surfaces, responsive navigation, polished cards, and route specific content that keeps the experience feeling deep from page to page.

Luxury visual system

Glassy panels, soft reflections, neon edge lighting, and balanced spacing create a high end interface with strong contrast and depth.

Route by route structure

Every major page lives in its own folder with an index.html entry so all internal links remain static friendly on GitHub Pages.

Animated interaction design

Buttons shimmer, cards tilt, reveals stage in as you scroll, and the navigation adapts smoothly on mobile and desktop.

Reusable front end system

Shared CSS and JavaScript make the project easier to maintain while still giving every route a strong page specific feel.
